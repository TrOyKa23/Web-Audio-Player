ANTINODE WEB MUSIC PLAYER 

==========================
Описание
==========================
ANTINODE — современный веб-плеер для музыки с красивой Welcome-страницей, поддержкой загрузки треков, бесконечным плейлистом и параметрическим эквалайзером. Проект состоит из двух частей: Welcome Page (стартовая страница) и самого плеера (работает на Node.js + Express).

==========================
Как запустить проект
==========================

1. Скачайте или склонируйте репозиторий на свой ПК.

2. Установите Node.js  
   Скачать: https://nodejs.org/

3. Установите зависимости для плеера  
   Откройте терминал и перейдите в папку плеера:
   cd "Player v1.2"
   npm install

4. Запустите сервер плеера  
   npm start  
   После запуска сервер будет доступен по адресу:  
   http://localhost:3000

5. Откройте Welcome Page  
   Откройте файл  
   Player v1.2/Wellcome page/index.html  
   в браузере (двойной клик или через "Открыть с помощью...").

   Либо используйте любой статический сервер (например, Live Server для VS Code).

6. Переход к плееру  
   На Welcome Page есть кнопка "Demo", которая откроет плеер в новой вкладке.

==========================
Технологии и стек
==========================

- Welcome Page:  
  HTML, CSS, JS (без фреймворков), видео-фон, кастомный курсор, анимации.

- Плеер:  
  Node.js, Express, Multer (загрузка файлов), чистый JS, HTML5 Audio API, Canvas (эквалайзер), адаптивная верстка.

==========================
Фичи проекта
==========================

Welcome Page:
- Красивый видео-фон (loop)
- Минималистичный дизайн
- Кастомный курсор
- Анимированная стрелка для скролла
- Кнопка перехода к плееру

Плеер:
- Загрузка треков (drag&drop или кнопка)
- Бесконечный плейлист
- Управление воспроизведением (play/pause, next, prev, repeat, shuffle)
- Громкость и mute
- Параметрический эквалайзер (6 полос, визуализация на Canvas)
- Сохранение загруженных треков на сервере
- Адаптивный интерфейс

==========================
Как работает проект
==========================

- Welcome Page — отдельная стартовая страница, открывается как обычный HTML-файл или через статический сервер.
- Player — отдельное приложение на Node.js, запускается через npm start и доступно по адресу http://localhost:3000.
- Переход с Welcome Page на плеер осуществляется по кнопке "Demo".
